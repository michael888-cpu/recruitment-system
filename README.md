# 招聘面试管理系统

## 功能说明

### 1. 题库管理
- **大类划分**：社会招聘 / 校园招聘
- **岗位管理**：在大类下创建岗位（包含序列、岗位名称）
- **按岗位划分题库**：每个岗位有独立的题库，可编辑/增删改

### 2. 面试链接
- 创建可复制的面试链接
- 每次邀请生成唯一链接
- 邀请记录展示（含面试链接）

### 3. 评分系统
- **岗位级别评分维度**：不同岗位配置不同评分标准
- **题目级别评分**：每道题有独立评分标准
- **面试回答记录**：存储候选人回答，供AI分析

### 4. 候选人端
- 通过面试链接访问
- 填写个人信息
- 参与面试答题

## 技术栈

- **后端**：Node.js + Express + SQLite
- **前端**：React
- **数据库**：SQLite

## 快速开始

```bash
npm install
npm run init-db
npm run dev
```

- 后端地址：http://localhost:3001
- 前端地址：http://localhost:3000
