# 招聘面试管理系统 - 部署指南

## 第一步：在 Supabase 创建数据库

1. 打开 https://supabase.com/dashboard
2. 进入您的"招聘面试系统"项目
3. 点击左侧 **SQL Editor**
4. 复制 `supabase-schema.sql` 文件中的内容粘贴进去
5. 点击 **Run** 执行

## 第二步：获取 Supabase 配置

1. 在 Supabase 控制台，点击左侧 **Project Settings**
2. 点击 **API**
3. 复制以下两个值：
   - **Project URL**: 如 `https://xxxxx.supabase.co`
   - **anon public key**: 一长串字符

## 第三步：修改配置文件

打开 `js/supabase-config.js`，将第一行的配置替换为您的实际值：

```javascript
const SUPABASE_URL = '您的Project URL';
const SUPABASE_ANON_KEY = '您的anon key';
```

## 第四步：部署到云端

### 方案A：腾讯云 Pages（推荐，国内访问快）

1. 将整个项目文件夹上传到 GitHub
2. 登录腾讯云 Pages，选择 GitHub 仓库
3. 部署即可

### 方案B：Netlify

1. 访问 https://netlify.com
2. 用 GitHub 登录
3. 选择"Add new site" → "Deploy manually"
4. 拖入整个项目文件夹
5. 获取网址

## 访问地址

- 管理后台: `部署网址/index.html`
- 候选人端: `部署网址/interview.html?link=面试链接`
